# End-to-End Blog Creation Suite

A comprehensive AI-powered blog creation system with 169 specialized agents across 18 phases.

## Quick Start

1. Install Python 3.10+
2. Install dependencies: `pip install -r requirements.txt`
3. Get free Pexels API key from https://www.pexels.com/api/
4. Create `.env` file: `PEXELS_API_KEY=your_key_here`
5. Run: `streamlit run main.py`
6. Open browser: http://localhost:8501

## Features

- 169 AI agents across 18 phases
- Complete blog automation
- SEO optimization 
- Pexels image integration
- Quality assurance
- Analytics dashboard
- Free/open-source tools only

## Project Structure

- `main.py` - Streamlit app
- `agents/` - AI agent implementations
- `utils/` - Database and API utilities
- `config/` - Settings and configuration
- `data/` - SQLite database and files

## Usage

1. Navigate to "Blog Creation" tab
2. Enter topic and preferences
3. Click "Create Blog" 
4. Monitor progress through 9 phases
5. Review generated content

Built with Python 3.10 + Streamlit + 169 AI Agents
